How to Install Postgres Locally on MacOS with Homebrew

PostgreSQL is a powerful, open-source relational database system. Installing it on macOS using Homebrew is one of the easiest and most flexible ways to get started. This guide walks you through the process.

Step 1: Install Homebrew (If You Haven’t Already)

Homebrew is a package manager for macOS. If you don’t have it yet, you can install it by running the following command in your terminal: 

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Step 2: Install PostgreSQL

To install PostgreSQL using Homebrew, simply run: 

brew install postgresql

This will download and install the latest version of PostgreSQL available via Homebrew.

Step 3: Start the PostgreSQL Service

Once installed, you can start the PostgreSQL server using: 

brew services start postgresql

Step 4: Verify the Installation

You can confirm PostgreSQL is working by checking the version: 

psql --version

And then try launching the PostgreSQL shell: 

psql postgres

Step 5: Create a Database

You can create a new database using: 

createdb my_database_name

Replace my_database_name with the name of your desired database.

Troubleshooting Tips

  • Permission Issues: Use sudo cautiously if you hit permission errors.
  • Service Doesn’t Start: Try brew services restart postgresql.
  • Port Conflicts: Ensure no other database service is running on port 5432.

That’s it! PostgreSQL should now be installed and running on your Mac. You’re ready to start developing locally with a reliable relational database.
